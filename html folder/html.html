HTML Notes

    Structure

    • <!DOCTYPE html>: Declares the document type (HTML5).
    • <html>: Root element of an HTML document.
    • <head>: Contains metadata (not displayed on the page).
    • <body>: Contains the visible content of the page.

    Headings

    • <h1> to <h6>: Heading levels (1 being most important).

    Paragraphs and Text

    • <p>: Defines a paragraph.
    • <br>: Inserts a line break.
    • <strong>: Bold text.
    • <em>: Emphasized text (usually italic).

    Lists

    • <ul>: Unordered list (bullet points).
    • <ol>: Ordered list (numbered).

    Links

    • <a>: Creates a hyperlink.

    Images

    • <img>: Inserts an image.

    Tables

    • <table>: Defines a table.

    Forms

    • <form>: Creates a form for user input.

    Common Attributes

    • id: Unique identifier for an element.
    • class: Assigns a CSS class to style elements.
    • style: Applies inline CSS styles.
    • src: Specifies the source of an external resource (e.g., image).
    • alt: Provides alternative text for images or other elements.
